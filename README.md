About
-----
A sample application for FaceBook that runs on the RUN@cloud platform provided by CloudBees.

Technology
----------

This application uses:
  Apache MyFaces JSF 2.1
  Apache WebBeans CDI
  Apache MyFaces CODI extensions to CDI
  RestFB
